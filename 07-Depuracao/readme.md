# Depuração JavaScript

1. #### Exception
Definição: QUando é encontrado um erro no código, assim gerando uma exceção. Caso não for tratada o programa pode parar.

2. #### Try..Catch
Definição: Permite capturar exceções através do try no código, se houver uma exceção no código podemos tratar com o catch que vai nos exibir essa exceção.

3. #### Finally 
Definição:Sendo executado sempre após o catch, mas não tendo influência sobre eles. Útil para exibir uma mensagem final

4. #### Throw
Definição: Podendo criar nossas exceções no código, caso alguma parte precisa ser verificada e queremos exibir o erro para um parceiro de equipe.

5. #### Assertion
Definição: Entrando mais em vlaidações, trabalhando com condições verificando se são verdadeiras, se forem falsas gera um erro

6. #### Debugger
Definição: Ajuda a debugar o código, mostrando etapa por até até o debugger aparecer ou procurar outro debugger
